

# بخش ۱: پیکربندی VS Code 

برای تبدیل شدن به یک برنامه‌نویس حرفه‌ای، محیط توسعه (IDE) شما باید هوشمند و کارآمد باشد. در این بخش ابزارهای کلیدی (VS Code) را تنظیم می‌کنیم.

### ۱. افزونه‌های ضروری (Extensions)
وارد بخش Extensions (`Ctrl + Shift + X`) شوید و افزونه‌های زیر را نصب کنید:
* **پکیج Python:** برای برجسته‌سازی سینتکس، عیب‌یابی (Debugging) و تکمیل خودکار کد (IntelliSense).
* **پکیج Pylance:** موتور تحلیل کد فوق‌العاده سریع مایکروسافت که خطاهای نگارشی و تایپی پایتون را قبل از اجرا به شما نشان می‌دهد.
* **پکیج GitLens:** قدرتمندترین ابزار برای مشاهده تاریخچه گیت. به شما نشان می‌دهد هر خط کد را چه کسی، در چه تاریخی و با چه کامیتی تغییر داده است.
